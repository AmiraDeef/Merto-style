# 🛒 Merto Shop - Premium E-Commerce UI Clone

A comprehensive, high-fidelity recreation of the **Merto Shop** e-commerce interface. This project was a deep-dive challenge into complex **Bootstrap 5** layouts and **Vanilla JavaScript** interactivity.

---
* **Inspiration:** [Merto Shop Demo](https://demo.theme-sky.com/merto/shop/apple-watch-series-9-gps-45mm/)

## 🚀 Key Technical Challenges
This wasn't just a simple landing page. This "huge task" involved:

* **Complex Grid Architecture:** Rebuilding the multi-column product layout and the sophisticated dark footer seen in the Merto theme.
* **Interactive UI Components:** * **Dynamic Countdown Timer:** A functional JS-driven "Limited Time Offer" clock.
    * **Image Switcher:** Logic to swap main product views based on thumbnail selection.
    * **Quantity Logic:** Custom increment/decrement selectors for the shopping cart.
* **Tabbed Specifications:** Managing the "Description | Specifications | Reviews" toggle system.
* **Responsive Typography:** Matching the clean, minimalist Apple-style aesthetic across all screen sizes.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for high-end SEO.
* **Bootstrap 5:** Extensive use of Flexbox, Grids, and Utility classes.
* **Vanilla JavaScript:** Custom logic for all interactive shop elements.
* **CSS3:** Custom styling for the "In Stock" badges, price formatting, and hover effects.
